# Autopilot Quick Reference

## Daily Workflow

1. **Create content in Claude**
   ```
   Write engaging post about [topic]
   Platform: LinkedIn
   Tone: Professional yet conversational
   ```

2. **Open Autopilot**
   ```
   https://aalyeldabaa-web.github.io/autopilot/
   ```

3. **Schedule Post**
   - Click **+ Schedule Post**
   - Paste content
   - Select platforms
   - Set date/time
   - Click **🚀 Schedule**

4. **Done!**
   - Autopilot handles everything
   - Post publishes automatically
   - Check GitHub Actions for confirmation

---

## Platform Selection Guide

**LinkedIn Personal** - Thought leadership, professional insights  
**LinkedIn Company** - Ajax announcements, company updates  
**Facebook Personal** - Community building, personal stories  
**Facebook Page** - Ajax marketing, client testimonials  

---

## Best Practices

✅ **Schedule ahead** - Queue 1-2 weeks of content  
✅ **Mix platforms** - Don't post same content everywhere  
✅ **Peak times** - LinkedIn 8AM-10AM, Facebook 1PM-3PM  
✅ **Review queue** - Check scheduled posts weekly  
✅ **Monitor results** - Check GitHub Actions logs

## Common Tasks

### Check Scheduled Posts
```
Open Autopilot → View "Scheduled Posts" section
```

### Edit a Post
```
Click post card → Edit → Save Changes
```

### Delete a Post
```
Click post → Edit → Delete button
```

### View Published Posts
```
Posts marked with ✓ PUBLISHED
Check GitHub Actions logs for links
```

## Troubleshooting

**Post didn't publish?**
→ GitHub Actions tab → View latest run → Check logs

**Can't schedule post?**
→ Refresh page → Check internet connection

**Token expired?**
→ See SETUP.md Part 2/3 → Regenerate tokens
